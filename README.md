# Experimental 444mhz Plugin
Experimental plugin for overclocking to 444mhz.

## Usage
L_TRIGGER + R_TRIGGER + SELECT to switch between 333 and 444mhz.

## Description
This is an experimental project for technical/educational purposes only, intended for developers only. Stability is not guaranteed, use with caution.

## Compatibility
Tested only on PSP Slim.
